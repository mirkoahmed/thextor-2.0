# thextor 2.0

A lightweight, fast, stable, simple, and essential text editor.

# Flavors
This project features two build variants:
- *Thextor*: The original dark themed version with one of the best combination of colors for you eyes health.
- *Thextor Ink*: The alternative light themed version, ink on paper, for the best reading contrast.

# Building
This project uses the Gradle build system. You can build the different flavors using Android Studio or via the command line:

- To build Thextor: `./gradlew assembleStandardRelease`
- To build Thextor Ink: `./gradlew assembleInkRelease `
